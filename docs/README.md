# Flutter SNS App ドキュメント

このディレクトリには、Flutter SNS Appプロジェクトのセットアップと開発に関するドキュメントが含まれています。

|         ドキュメント         |              内容              | 対象環境 | 所要時間 |
| ---------------------------- | ------------------------------ | -------- | -------- |
| [SETUP-01.md](./SETUP-01.md) | Flutter開発環境のセットアップ  | Windows  | 約30分   |
| [SETUP-02.md](./SETUP-02.md) | Android Emulatorのセットアップ | Windows  | 約45分   |

## 📖 各セットアップガイドの詳細

### SETUP-01.md - Flutter開発環境のセットアップ

Flutter開発に必要な基本ツールとSDKのインストール手順を説明しています。

|          セクション           |                        内容                        |
| ----------------------------- | -------------------------------------------------- |
| **前提条件**                  | Git for Windows、Visual Studio Codeのインストール  |
| **Flutter SDKのインストール** | VS Code拡張機能を使用したFlutter SDKのインストール |
| **動作確認**                  | 新規プロジェクトの作成とWebブラウザでの実行テスト  |
| **トラブルシューティング**    | よくある問題と解決方法                             |

### SETUP-02.md - Android Emulatorのセットアップ

Android開発のためのエミュレータ環境を構築する手順を説明しています。

|            セクション            |                     内容                     |
| -------------------------------- | -------------------------------------------- |
| **Android Studioのインストール** | Android Studio本体とSDKツールのセットアップ  |
| **Androidライセンスへの同意**    | `flutter doctor --android-licenses`の実行    |
| **VMアクセラレーションの有効化** | BIOSとWindowsの仮想化機能の設定              |
| **エミュレータの作成**           | Device Managerを使用した仮想デバイスの作成   |
| **コマンドライン起動**           | 環境変数の設定とコマンドラインからの起動方法 |
| **セットアップの検証**           | `flutter doctor`と`flutter devices`での確認  |

### SETUP-03.md - アプリケーション起動手順

flutter_sns_appプロジェクトの起動方法を説明しています。

|         セクション         |                      内容                      |
| -------------------------- | ---------------------------------------------- |
| **コマンドライン起動**     | `flutter run`を使用した起動手順                |
| **VS Code起動**            | VS Codeのデバッグ機能を使用した起動方法        |
| **起動後の操作**           | ホットリロード、ホットリスタートなどの操作方法 |
| **トラブルシューティング** | ビルドエラーやデバイス認識の問題の解決方法     |

## 💡 Tips

- すべてのセットアップが完了したら、`flutter doctor`コマンドですべての項目にチェックマークが付いていることを確認してください
- エミュレータの起動が遅い場合は、グラフィックアクセラレーションの設定を確認してください
- VS Codeを使用すると、デバッグやホットリロードがより便利に利用できます
